# Getting Help

Support for Zirco is available in the following channels:

- **Matrix** (preferred): Join us at [#zirco:zirco.dev](https://matrix.to/#/#zirco:zirco.dev)
  for real-time discussion and support.
- **Discord**: Join [LogN's Discord server](https://discord.gg/hyxpPWxpWr) in the `/home/zirco`
  category. This server is not as regularly monitored as Matrix.
- **GitHub Issues**: For bug reports and feature requests, please open an issue on the project's
  [GitHub repository](https://github.com/zirco-lang/zrc/issues). This is not intended for general
  support, but we will do our best to help if you have trouble.
